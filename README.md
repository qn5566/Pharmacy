# PharmacyKotlin APP
API - 使用<a href="https://data.gov.tw/">政府資料開放平臺</a> 



特色
1.使用ViewBinding做面向開發
2.filter和groupBy的使用
3.Recyclerview的使用
4.Google Map開發
5.GoogleMap.InfoWindowAdapter客製化使用
6.抓取使用者位置坐區域探索


使用套件
1.okhttp - 4.9.0
2.gson - 2.8.6
3.recyclerview - 1.1.0
4.play-services-location 17.0.0
5.play-services-maps 17.0.0
